## Built With
- HTML, CSS
- Flexbox, Grid
- VSCode
- Node.js
- Express.js
- OpenWeatherMap API

## Deployment
1) Clone or Download this repo.
2) Change directory to the repo.
3) Run `npm install` to install dependencies.
4) Run `node server` or `npm run start`.
5) Open `http://localhost:3000/` on your web browser.
6) Enjoy :)
